
## 🛠️ Technologies Used

- React Toastify

## 💻 Getting Started

### Prerequisites

- Node.js (v16 or above recommended)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/strictlyManish/react-ecommerce.git

# Navigate to the project directory
cd react-ecommerce

# Install dependencies
npm install

# Start the development server
npm run dev
